# 希尔排序

希尔排序是对插入排序的一个改进版本，比简单插入排序有更高的效率，也称为缩小增量排序

希尔排序的实现方法：希尔排序采用增量分组的方法将元素划分为若干组，然后对每个分组进行插入排序，随后逐步缩小增量，继续重复分组和对分组进行插入排序，当增量缩小为1时，序列变为大部分有序，只需要微调不需要频繁移动

希尔排序的增量序列选择直接影响到排序性能


希尔排序是一种不稳定的排序方法
